## 🚀 基本使用

### 方法 1: 使用智能查询脚本（推荐）

1. 编辑 `scripts/intelligent_judicial_query.py` 的 `main()` 函数:

```python
def main():
    # 配置查询信息
    name = "目标姓名"
    id_card = "目标身份证号"

    # 查询配置
    headless = False  # 是否无头模式（False 可以看到浏览器操作）
    max_retries = 3  # 最大重试次数
    include_court_docs = False  # 是否查询裁判文书（需要处理验证码）

    # 创建查询器
    query = IntelligentJudicialQuery(headless=headless, max_retries=max_retries)

    # 执行查询
    results = query.query_all(name, id_card, include_court_docs=include_court_docs)
```

2. 运行脚本:

```bash
python scripts/intelligent_judicial_query.py
```

3. 查看结果:

结果保存在 `reports/智能司法查询结果.json`

## 🔧 配置验证码识别（可选）

如果需要自动识别验证码，编辑 `references/captcha_api_config.json`:

```json
{
  "baidu": {
    "enabled": true,
    "api_key": "YOUR_BAIDU_API_KEY",
    "secret_key": "YOUR_BAIDU_SECRET_KEY"
  }
}
```

**获取 API Key**:
- 百度: https://cloud.baidu.com/product/ocr/general
- 阿里云: https://www.aliyun.com/product/ocr
- 腾讯云: https://cloud.tencent.com/product/ocr

## 📝 查询配置说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `headless` | 是否无头模式（False 可以看到浏览器操作） | False |
| `max_retries` | 每个策略的最大重试次数 | 3 |
| `include_court_docs` | 是否查询裁判文书（需要处理验证码） | False |

## ❓ 常见问题

### Q: 如何查询裁判文书？

A: 设置 `include_court_docs=True` 并准备好处理验证码（或配置 OCR API）。

### Q: 验证码怎么办？

A:
1. 配置第三方 OCR API（推荐）
2. 或使用人工输入模式（当检测到验证码时暂停，等待人工输入）

### Q: 查询速度慢怎么办？

A:
- 启用无头模式：`headless=True`
- 减少重试次数：`max_retries=1`
- 不查询裁判文书：`include_court_docs=False`

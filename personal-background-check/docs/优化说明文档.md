本次优化主要针对司法平台爬虫逻辑进行了全面升级，特别是裁判文书查询功能。优化内容包括：

### 2. 验证码识别和处理模块

**文件**: `scripts/captcha_handler.py`

**功能特性**:
- 自动检测页面中的验证码元素
- 截取验证码图片
- 支持第三方 OCR API（百度、阿里云、腾讯云）
- 人工验证码输入支持
- 验证码识别结果缓存（用于后续训练）

**支持的 API 提供商**:

| 提供商 | 安装命令 | 申请地址 |
|--------|----------|----------|
| 百度 | `pip install baidu-aip` | https://cloud.baidu.com/product/ocr/general |
| 阿里云 | `pip install aliyun-python-sdk-core aliyun-python-sdk-ocr` | https://www.aliyun.com/product/ocr |
| 腾讯云 | `pip install tencentcloud-sdk-python` | https://cloud.tencent.com/product/ocr |

**配置文件**: `references/captcha_api_config.json`

```json
{
  "baidu": {
    "enabled": true,
    "api_key": "YOUR_API_KEY",
    "secret_key": "YOUR_SECRET_KEY"
  },
  "aliyun": {
    "enabled": false,
    "access_key_id": "YOUR_ACCESS_KEY_ID",
    "access_key_secret": "YOUR_ACCESS_KEY_SECRET"
  },
  "tencent": {
    "enabled": false,
    "secret_id": "YOUR_SECRET_ID",
    "secret_key": "YOUR_SECRET_KEY"
  }
}
```

**使用方法**:
```python
from scripts.captcha_handler import CaptchaHandler

# 创建验证码处理器
handler = CaptchaHandler(driver=webdriver_instance)

# 自动解决验证码
code = handler.solve_captcha(auto_retry=True)

# 人工输入验证码
code = handler.manual_solve()
```

### 3. 反爬虫策略模块

**文件**: `scripts/anti_crawler_strategy.py`

**功能特性**:
- **隐身模式策略**: 使用 CDP 命令隐藏 webdriver 特征
- **人类行为模拟**: 随机打字、滚动、点击延迟
- **请求节流**: 控制请求频率，避免触发限制
- **重试策略**: 指数退避重试机制
- **策略管理器**: 组合多种策略

**支持的策略**:

| 策略名称 | 说明 |
|----------|------|
| StealthModeStrategy | 隐藏自动化特征 |
| HumanBehaviorStrategy | 模拟人类行为 |
| RequestThrottlingStrategy | 请求节流 |
| RetryStrategy | 自动重试 |
| ProxyRotationStrategy | 代理轮换（需配置） |
| UserAgentRotationStrategy | User-Agent 轮换 |

**使用方法**:
```python
from scripts.anti_crawler_strategy import create_default_strategy_manager

# 创建策略管理器
manager = create_default_strategy_manager(driver)

# 设置所有策略
manager.setup_all()

# 请求前执行策略
manager.before_request_all()

# 请求后执行策略
manager.after_request_all()
```

## 📖 使用指南

### 快速开始

#### 1. 安装依赖

```bash
# 基础依赖（必需）
pip install -r requirements.txt

# 验证码识别 API（可选）
pip install baidu-aip  # 百度 OCR
# 或
pip install aliyun-python-sdk-core aliyun-python-sdk-ocr  # 阿里云 OCR
# 或
pip install tencentcloud-sdk-python  # 腾讯云 OCR
```

#### 2. 配置验证码 API（可选）

编辑 `references/captcha_api_config.json`:

```json
{
  "baidu": {
    "enabled": true,
    "api_key": "你的百度 API Key",
    "secret_key": "你的百度 Secret Key"
  }
}
```

#### 3. 执行查询

**方式 1: 使用智能查询脚本（推荐）**

编辑 `scripts/intelligent_judicial_query.py` 的 main 函数:

```python
def main():
    # 配置查询信息
    name = "目标姓名"
    id_card = "目标身份证号"

    # 查询配置
    headless = False  # 是否无头模式
    max_retries = 3  # 最大重试次数
    include_court_docs = True  # 是否查询裁判文书

    # 创建查询器
    query = IntelligentJudicialQuery(headless=headless, max_retries=max_retries)

    # 执行查询
    results = query.query_all(name, id_card, include_court_docs=include_court_docs)
```

运行脚本:

```bash
python scripts/intelligent_judicial_query.py
```

**方式 2: 使用独立的裁判文书查询脚本**

```python
from scripts.court_documents_selenium_query import query_court_documents_selenium

result = query_court_documents_selenium(
    name="张三",
    id_card="110101199001011234",
    headless=False
)

print(result)
```

## ❓ 常见问题

### Q1: 为什么裁判文书查询默认不开启？

**A**: 裁判文书网有严格的反爬虫机制，可能触发验证码。默认关闭是为了避免查询过程中被打断。如果需要查询，请设置 `include_court_docs=True` 并准备好处理验证码。

### Q2: 如何处理验证码？

**A**: 有两种方式：
1. **自动识别**: 配置第三方 OCR API（推荐）
2. **人工输入**: 当检测到验证码时，脚本会暂停，等待人工输入

### Q3: undetected-chromedriver 安装失败怎么办？

**A**: 可以使用标准 Selenium，脚本会自动回退到标准策略。确保安装了 Chrome 浏览器和对应的 ChromeDriver。

### Q4: 查询速度慢怎么办？

**A**:
- 启用无头模式：`headless=True`
- 减少重试次数：`max_retries=1`
- 减少查询平台：不查询裁判文书 `include_court_docs=False`

### Q5: 如何查看查询结果？

**A**: 查询结果保存在 `reports/` 目录下，以 JSON 格式存储。可以使用任何文本编辑器或 JSON 查看工具打开。

### Q6: 验证码识别不准确怎么办？

**A**:
1. 切换到其他 OCR 提供商（百度、阿里云、腾讯云）
2. 使用人工输入模式
3. 验证码识别失败后，脚本会自动切换到人工输入

### Q7: 可以使用代理吗？

**A**: 可以，但需要配置。参考 `ProxyRotationStrategy` 类，在创建 driver 时添加代理设置。

### Q8: 如何避免被反爬虫系统检测？

**A**:
- 使用 `undetected-chromedriver`（策略 1）
- 启用隐身模式策略
- 模拟人类行为（随机延迟、打字、滚动）
- 控制请求频率
- 使用代理轮换

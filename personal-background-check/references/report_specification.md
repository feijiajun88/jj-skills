# 报告格式规范

本文档详细定义个人背景调查报告的格式规范、字段说明和输出标准。

---

## 一、报告版本

- **版本号**：v1.0.0
- **生效日期**：2025-02-02
- **适用范围**：个人背景调查技能生成的所有报告

---

## 二、报告结构规范

### 报告组成

报告由以下7个主要部分组成：

1. 报告概览
2. 司法信息
3. 信用与税务信息
4. 交通违法信息
5. 不动产与资产信息
6. 综合风险评估
7. 附录

### 文档属性

| 属性 | 要求 |
|------|------|
| 文件格式 | DOCX / PDF / Markdown |
| 编码格式 | UTF-8 |
| 默认字体 | 中文字体：宋体，英文字体：Times New Roman |
| 正文字号 | 小四号（12pt）|
| 标题字号 | 一级标题：二号（22pt），二级标题：三号（16pt），三级标题：四号（14pt）|
| 页边距 | 上下2.54cm，左右3.17cm |
| 行距 | 1.5倍行距 |

## 五、数据脱敏规则

### 身份证号脱敏
```
原始：110101199001011234
脱敏：110***********1234
规则：保留前3位和后4位，中间用*代替
```

### 手机号脱敏
```
原始：13812345678
脱敏：138****5678
规则：保留前3位和后4位，中间用*代替
```

### 银行卡号脱敏
```
原始：6222021234567890123
脱敏：6222***********0123
规则：保留前4位和后4位，中间用*代替
```

### 地址脱敏
```
原始：北京市朝阳区XX路XX号XX小区X栋X单元XXX室
脱敏：北京市朝阳区XX路XX号XX小区X栋X单元***室
规则：保留到楼栋单元，门牌号用*代替
```

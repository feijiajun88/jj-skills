# 行业调查指引技能使用说明

欢迎使用普道小微贷款行业调查指引技能！

---

## 技能简介

本技能为普道小微贷款信贷员提供**贷前行业调查**的标准化指引和报告生成工具。

### 核心功能

1. **行业知识库** - 内置制造业、批发零售、餐饮、建筑等常见行业的调查要点
2. **实时联网搜索** - 获取最新行业政策、市场数据、竞争格局
3. **调查清单生成** - 根据行业特征自动生成现场调查要点
4. **结构化报告输出** - 生成标准Markdown格式的行业调查报告

---

## 快速开始

### 基本用法

```
输入：请帮我调查[行业名称]的行业情况

示例：
- 请帮我调查制造业的行业情况
- 我想了解餐饮行业的风险特征
- 批发零售业需要关注哪些调查要点
```

### 高级用法

```
输入：请帮我调查[行业名称]，重点关注[特定方面]

示例：
- 请帮我调查餐饮业，重点关注疫情后恢复情况
- 调查制造业，重点关注环保政策和用工成本
- 调查装修行业，重点关注垫资情况和回款风险
```

## 报告输出

### 报告结构

生成的报告包含以下模块：

1. **行业概况** - 市场规模、发展趋势、竞争格局、政策环境
2. **风险特征分析** - 行业周期性、主要风险、政策风险
3. **信贷调查要点** - 现场调查、财务核查、交叉验证
4. **关键财务指标参考** - 毛利率、周转率、偿债能力
5. **风险提示与建议** - 重点风险、调查建议、贷后管理

### 报告格式

- Markdown格式，便于阅读和归档
- 表格化展示数据，清晰直观
- 风险等级用颜色标记（🔴高风险、🟠中高风险、🟡中等风险）

## 注意事项

1. **报告仅供参考** - 本技能生成的报告仅供参考，实际调查需结合客户具体情况

2. **持续更新** - 行业情况变化较快，建议定期更新内置知识库

3. **数据时效性** - 联网搜索的信息可能存在滞后，需注意数据发布时间

4. **地区差异** - 同一行业在不同地区可能存在显著差异，需考虑地域因素

5. **保密要求** - 使用联网搜索时注意保护客户隐私和商业机密

## 常见问题

### Q1：报告中的数据准确吗？

A：报告数据来源于官方统计和行业研究机构，具有一定权威性。但由于行业情况变化较快，建议结合实地调查和客户实际情况进行综合判断。

### Q2：如何选择行业模板？

A：根据客户主营业务选择对应的行业模板。如果客户涉及多个行业，建议选择占比最高的行业，并在调查中兼顾其他行业特点。

### Q3：指标参考值如何使用？

A：指标参考值为行业平均水平，用于对比分析。实际应用时，需要考虑地域差异、规模差异、个体差异等因素，不能作为绝对标准。

### Q4：如何更新知识库？

A：建议每季度更新一次行业指标。可以从国家统计局、行业协会、行业研究报告等渠道获取最新数据。

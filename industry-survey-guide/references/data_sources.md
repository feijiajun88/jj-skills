## 官方统计数据

### 1.1 国家统计局

**网站：** http://www.stats.gov.cn/

**数据内容：**
- 国民经济和社会发展统计公报
- 各行业统计数据
- 分地区经济数据
- 价格指数、生产指数等

**数据特点：**
- ✅ 权威性高
- ✅ 覆盖面广
- ⏰ 更新周期：月度、季度、年度
- 📊 数据格式：表格、图表、统计公报

**使用场景：**
- 行业整体规模查询
- 行业发展趋势分析
- 地区经济数据对比
- 价格指数查询

**注意事项：**
- 统计数据通常有1-2个月的滞后
- 部分细分行业数据可能缺失
- 官方统计口径可能与实际业务有差异

**使用场景：**

### 1.3 银保监会

**数据内容：**

## 数据可靠性评估

### 7.1 数据来源可靠性分级

| 可靠性等级 | 数据来源 | 特点 |
|------------|----------|------|
| 🔴 最高 | 官方统计、监管机构 | 权威、可信 |
| 🟠 高 | 行业协会、上市公司 | 较权威、可信度高 |
| 🟡 中 | 知名研究机构、财经媒体 | 专业性强、需验证 |
| 🔵 一般 | 商业平台、网络信息 | 需要交叉验证 |

### 7.2 数据时效性评估

| 时效性 | 数据类型 | 适用场景 |
|--------|----------|----------|
| ⚡ 实时 | 股价、新闻 | 市场动态跟踪 |
| 📅 月度 | 宏观数据、行业数据 | 趋势分析 |
| 📊 季度 | 企业财报、监管数据 | 经营分析 |
| 📆 年度 | 统计公报、行业报告 | 长期趋势 |

### 7.3 数据质量评估标准

| 评估维度 | 评估标准 | 权重 |
|----------|----------|------|
| 权威性 | 数据来源的权威程度 | 30% |
| 时效性 | 数据的时效性 | 25% |
| 准确性 | 数据的准确性 | 25% |
| 完整性 | 数据的完整性 | 10% |
| 相关性 | 与调查目的的相关性 | 10% |

## 使用建议

### 8.1 数据选择原则

1. **优先使用官方数据**
   - 国家统计局、央行、银保监会等官方数据优先
   - 行业协会数据次之
   - 商业数据需验证

2. **关注数据时效性**
   - 优先使用最新数据
   - 注意数据更新周期
   - 避免使用过时数据

3. **多源数据验证**
   - 同一数据至少从2个来源验证
   - 关注数据来源的一致性
   - 发现差异需深入调查

4. **注意数据口径**
   - 关注数据统计口径
   - 不同来源数据可能口径不同
   - 使用时注意区分

### 8.2 联网搜索策略

1. **搜索关键词**
   - 行业名称 + 市场规模
   - 行业名称 + 发展趋势
   - 行业名称 + 竞争格局
   - 行业名称 + 政策

2. **搜索来源过滤**
   - gov.cn：政府官网
   - org.cn：协会、组织
   - com.cn：商业网站（注意验证）
   - cn：中国本土网站

3. **时间范围设置**
   - 最近1年：政策、新闻
   - 最近3年：市场数据
   - 最近5年：长期趋势

4. **数据交叉验证**
   - 同一数据从不同来源验证
   - 关注数据发布时间
   - 对比多个来源的数据

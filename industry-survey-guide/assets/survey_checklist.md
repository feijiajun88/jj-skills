## 现场调查清单

### 一、经营情况调查

#### 1.1 经营实体信息

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 工商信息 | 营业执照、组织机构代码 | 注册时间、注册资本、经营范围、法人变更 |
| 经营年限 | 实际经营时间 | 行业经验、稳定性 |
| 经营场所 | 租赁合同、房产证 | 租期、租金、地段、面积、装修 |
| 从业人员 | 员工数量、岗位分布 | 员工稳定性、技能水平 |

#### 1.2 生产/经营规模

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 生产能力 | 设备、产能、开工率 | 产能利用率、设备状况 |
| 销售规模 | 日/月/年销售额 | 销售稳定性、季节性 |
| 库存情况 | 原材料、在产品、产成品 | 库存结构、周转、积压 |
| 客户群体 | 主要客户、客户数量 | 客户依赖度、稳定性 |

#### 1.3 上下游关系

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 供应商 | 供应商数量、主要供应商 | 依赖度、稳定性、账期 |
| 销售渠道 | 销售模式、主要客户 | 客户质量、回款情况、账期 |
| 合作协议 | 合同、协议 | 合同条款、责任义务、有效期 |

### 二、财务状况调查

#### 2.1 收入核实

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 销售收入 | 销售记录、发票 | 收入真实性、完整性 |
| 其他收入 | 其他业务收入 | 收入合理性 |
| 收入季节性 | 月度、季度收入 | 波动性、周期性 |

#### 2.2 成本核实

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 主营业务成本 | 材料成本、人工成本、制造费用 | 成本结构、变化趋势 |
| 期间费用 | 销售费用、管理费用、财务费用 | 费用合理性、异常增长 |
| 税费 | 纳税申报、缴税凭证 | 纳税合规性、与收入匹配度 |

#### 2.3 资产负债核实

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 货币资金 | 现金、银行存款 | 流动性、真实性 |
| 应收账款 | 余额、账龄、主要客户 | 回收风险、真实性 |
| 存货 | 原材料、在产品、产成品 | 跌价风险、积压情况 |
| 固定资产 | 原值、折旧、净值 | 抵押情况、折旧政策 |
| 负债总额 | 银行贷款、应付账款、民间借贷 | 偿债压力、隐性负债 |

### 三、经营风险调查

#### 3.1 市场风险

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 行业竞争 | 竞争对手、竞争程度 | 竞争优势、市场地位 |
| 产品/服务竞争力 | 产品质量、价格、差异化 | 核心竞争力、可持续性 |
| 客户依赖 | 前五大客户占比 | 客户集中度、依赖风险 |
| 市场变化 | 需求变化、价格波动 | 应对能力、抗风险能力 |

#### 3.2 经营风险

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 管理风险 | 管理层经验、能力 | 管理水平、决策能力 |
| 技术风险 | 技术先进性、研发能力 | 技术壁垒、更新换代 |
| 合规风险 | 许可证、资质、环保、消防 | 合规性、潜在处罚风险 |
| 突发风险 | 事故、灾害、事件 | 应对措施、保险 |

#### 3.3 财务风险

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 资金链 | 现金流、融资渠道 | 资金充裕度、融资能力 |
| 偿债能力 | 流动比率、速动比率 | 短期偿债能力、长期偿债能力 |
| 民间借贷 | 是否存在、规模、利率 | 隐性负债、高息风险 |
| 担保情况 | 对外担保、互保 | 担保风险、或有负债 |

### 四、个人/家庭情况调查

#### 4.1 实际控制人

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 基本情况 | 年龄、学历、籍贯 | 背景稳定性 |
| 从业经历 | 工作经历、创业经历 | 行业经验、经营能力 |
| 家庭情况 | 婚姻、子女、家庭开支 | 家庭稳定性、负担 |
| 声誉口碑 | 行业口碑、社会评价 | 商业信誉、道德风险 |

#### 4.2 家庭资产

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 房产 | 住宅、商业、工业 | 价值、抵押、变现能力 |
| 车辆 | 数量、型号、成色 | 价值、抵押 |
| 其他资产 | 金融资产、其他投资 | 价值、流动性 |
| 负债 | 房贷、车贷、其他 | 偿债压力 |

### 五、特殊事项调查

#### 5.1 合规性

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 营业执照 | 是否齐全、是否在有效期 | 合法经营 |
| 特殊许可 | 行业许可、资质证书 | 行业准入、专业资质 |
| 税务登记 | 纳税登记、一般纳税人/小规模 | 纳税合规 |
| 社保缴纳 | 员工社保缴纳 | 用工合规 |

#### 5.2 司法风险

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 诉讼记录 | 民事诉讼、刑事诉讼 | 纠纷类型、风险程度 |
| 执行记录 | 被执行人、失信被执行人 | 失信情况、执行能力 |
| 处罚记录 | 行政处罚、行业处罚 | 合规性、风险程度 |

#### 5.3 其他

| 调查项 | 调查内容 | 关注要点 |
|--------|----------|----------|
| 合作伙伴 | 主要合作伙伴 | 合作稳定性、风险传染 |
| 关联方 | 关联企业、关联交易 | 利益输送、风险传染 |
| 重大投资 | 对外投资、扩张计划 | 资金占用、风险分散 |

## 财务核查清单

### 一、真实性核查

#### 1.1 收入真实性

| 核查方法 | 核查内容 |
|----------|----------|
| 银行流水核对 | 销售收入与银行流水匹配度 |
| 发票核对 | 发票金额与收入金额匹配 |
| 销售合同 | 合同金额与收入金额匹配 |
| 进货核对 | 进货与销售逻辑匹配 |
| 电费/水费/气费 | 公用费用与生产经营规模匹配 |
| 人工核对 | 工资总额与员工人数、收入规模匹配 |

#### 1.2 资产真实性

| 核查方法 | 核查内容 |
|----------|----------|
| 现场盘点 | 存货、固定资产现场盘点 |
| 产权核对 | 房产证、车辆证、设备发票 |
| 账实核对 | 账面资产与实物资产核对 |
| 抵押核对 | 抵押登记与账面资产核对 |

#### 1.3 负债真实性

| 核查方法 | 核查内容 |
|----------|----------|
| 银行征信 | 银行贷款明细 |
| 信用报告 | 信用记录、负债情况 |
| 民间借贷 | 通过访谈、流水发现民间借贷 |
| 应付账款 | 供应商对账、合同核对 |

### 二、完整性核查

#### 2.1 收入完整性

| 核查方法 | 核查内容 |
|----------|----------|
| 全账户流水 | 查看所有账户流水，发现隐匿收入 |
| 个人账户 | 核对个人账户流水，发现混同收入 |
| 现金收付 | 评估现金交易比例，发现隐匿收入 |
| 外部数据 | 税务申报数据、行业数据比对 |

#### 2.2 负债完整性

| 核查方法 | 核查内容 |
|----------|----------|
| 征信查询 | 全部征信查询，发现隐性负债 |
| 流水分析 | 分析还款流水，发现隐性借贷 |
| 访谈核实 | 通过访谈发现民间借贷 |
| 合同核对 | 核对所有合同，发现表外负债 |

### 三、合理性核查

#### 3.1 收入合理性

| 核查方法 | 核查内容 |
|----------|----------|
| 同业对比 | 与同业收入规模、增长速度对比 |
| 纵向对比 | 与历史数据对比，分析变化合理性 |
| 利润率对比 | 毛利率、净利率与行业对比 |
| 现金流匹配 | 现金流入与收入匹配度 |

#### 3.2 成本费用合理性

| 核查方法 | 核查内容 |
|----------|----------|
| 成本结构分析 | 成本构成是否合理 |
| 费用率分析 | 费用占收入比例是否合理 |
| 同业对比 | 与同业成本费用对比 |
| 异常波动 | 成本费用异常波动核查 |

### 四、偿债能力核查

#### 4.1 流动性

| 核查方法 | 核查内容 |
|----------|----------|
| 流动比率 | 流动资产/流动负债 |
| 速动比率 | 速动资产/流动负债 |
| 现金比率 | 现金/流动负债 |
| 经营现金流 | 经营活动现金净流入 |

#### 4.2 偿债压力

| 核查方法 | 核查内容 |
|----------|----------|
| 资产负债率 | 负债总额/资产总额 |
| 有息负债比 | 有息负债/负债总额 |
| 利息保障倍数 | EBIT/利息支出 |
| 还款计划 | 未来还款计划与现金流匹配 |

## 交叉验证清单

### 一、经营数据交叉验证

#### 1.1 收入交叉验证

| 验证维度 | 验证方法 |
|----------|----------|
| 进销存匹配 | 进货+期初库存-销售=期末库存 |
| 银行流水 | 销售收入与银行流水匹配 |
| 税务数据 | 销售收入与税务申报数据匹配 |
| 工资成本 | 工资总额与收入规模匹配 |
| 公用费用 | 电费/水费/气费与生产规模匹配 |

#### 1.2 成本交叉验证

| 验证维度 | 验证方法 |
|----------|----------|
| 采购量与库存 | 采购量与库存增减匹配 |
| 成本与收入 | 成本率与行业标准对比 |
| 成本与产量 | 成本变动与产量变动匹配 |
| 采购价格 | 采购价格与市场价格对比 |

#### 1.3 资产交叉验证

| 验证维度 | 验证方法 |
|----------|----------|
| 存货盘点 | 账面存货与实物存货对比 |
| 固定资产 | 账面资产与现场资产对比 |
| 折旧计提 | 折旧年限与实际使用对比 |
| 产权登记 | 资产产权与账面资产对比 |

### 二、财务数据交叉验证

#### 2.1 三大报表勾稽关系

| 验证维度 | 验证方法 |
|----------|----------|
| 利润表与资产负债表 | 净利润与未分配利润变动匹配 |
| 资产负债表与现金流量表 | 现金变动与资产负债表现金科目匹配 |
| 利润表与现金流量表 | 净利润与经营现金流匹配 |
| 所有者权益变动表 | 与资产负债表所有者权益匹配 |

#### 2.2 报表与原始凭证

| 验证维度 | 验证方法 |
|----------|----------|
| 收入与发票 | 收入金额与发票金额对比 |
| 成本与采购单 | 成本金额与采购单对比 |
| 费用与发票 | 费用金额与发票对比 |
| 资产与产权证 | 资产金额与产权证对比 |

### 三、内外部数据交叉验证

#### 3.1 外部数据

| 验证维度 | 验证方法 |
|----------|----------|
| 税务申报 | 纳税申报与财务报表对比 |
| 征信报告 | 征信负债与财务报表负债对比 |
| 行业数据 | 客户数据与行业数据对比 |
| 公开信息 | 企业信息与公开信息对比 |

#### 3.2 第三方数据

| 验证维度 | 验证方法 |
|----------|----------|
| 供应商对账 | 应付账款与供应商对账单对比 |
| 客户对账 | 应收账款与客户对账单对比 |
| 银行对账 | 银行存款与银行对账单对比 |
| 租赁合同 | 租金支出与租赁合同对比 |

### 四、时间序列交叉验证

#### 4.1 纵向对比

| 验证维度 | 验证方法 |
|----------|----------|
| 收入趋势 | 月度/季度/年度收入变化趋势分析 |
| 成本趋势 | 成本变动趋势与收入趋势对比 |
| 利润趋势 | 利润变动趋势分析 |
| 资产趋势 | 资产变动趋势分析 |
| 负债趋势 | 负债变动趋势分析 |

#### 4.2 季节性分析

| 验证维度 | 验证方法 |
|----------|----------|
| 月度对比 | 同比、环比分析 |
| 季度对比 | 同比、环比分析 |
| 节假日影响 | 节假日前后数据对比 |
| 行业周期 | 行业周期性分析 |

## 风险预警指标

### 一、收入类预警指标

| 指标 | 预警值 | 说明 |
|------|--------|------|
| 收入连续下降 | 连续3个月下降 | 经营困难风险 |
| 收入大幅波动 | 波动幅度>50% | 不稳定风险 |
| 毛利率低于行业 | 低于行业均值10个百分点 | 成本控制问题 |
| 收现率低 | 收现率<80% | 应收账款回收风险 |

### 二、负债类预警指标

| 指标 | 预警值 | 说明 |
|------|--------|------|
| 资产负债率 | >70% | 偿债压力大 |
| 流动比率 | <1.5 | 流动性不足 |
| 速动比率 | <1 | 快速偿债能力弱 |
| 现金比率 | <0.2 | 现金流紧张 |

### 三、经营类预警指标

| 指标 | 预警值 | 说明 |
|------|--------|------|
| 存货周转天数 | >行业均值150% | 库存积压风险 |
| 应收账款周转天数 | >行业均值150% | 回款风险 |
| 员工流失率 | >30% | 管理问题 |
| 客户流失率 | >20% | 竞争力问题 |

### 四、合规类预警指标

| 指标 | 预警值 | 说明 |
|------|--------|------|
| 有诉讼记录 | 存在诉讼 | 纠纷风险 |
| 有执行记录 | 存在被执行 | 信用风险 |
| 有行政处罚 | 存在处罚 | 合规风险 |
| 许可证到期 | 已过期/3个月内到期 | 经营风险 |

### 五、现金流预警指标

| 指标 | 预警值 | 说明 |
|------|--------|------|
| 经营现金净流量为负 | 连续6个月为负 | 现金流断裂风险 |
| 投资现金流出大 | 投资占收入>50% | 资金占用风险 |
| 筹资现金流入少 | 无法覆盖流出 | 融资能力弱 |
| 现金余额 | <3个月支出 | 资金紧张 |

## 使用说明

1. **灵活选择** - 根据客户特点、行业特点，选择适用的调查要点
2. **重点突出** - 对高风险领域重点关注，深入调查
3. **多种方法** - 综合使用多种核查方法，提高调查质量
4. **持续更新** - 根据监管要求、市场变化，及时更新调查清单
5. **记录留痕** - 详细记录调查过程，形成完整的调查档案

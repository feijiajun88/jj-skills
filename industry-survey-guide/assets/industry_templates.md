## 制造业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 以生产加工为主，原材料采购-加工制造-产品销售 |
| 资本投入 | 固定资产投入大，厂房、设备占用资金多 |
| 资金周转 | 生产周期较长，资金占压较多，周转相对较慢 |
| 季节性 | 部分行业受季节影响明显（如服装、食品加工） |
| 政策敏感 | 环保政策、产业政策影响较大 |
| 员工规模 | 一般10人以上，技术工人依赖度高 |

### 主要风险点

1. **市场风险**
   - 产品价格波动风险
   - 订单稳定性风险
   - 行业竞争加剧
   - 需求萎缩风险

2. **经营风险**
   - 原材料价格波动
   - 设备维护与更新
   - 生产安全事故
   - 劳动力成本上升

3. **政策风险**
   - 环保政策收紧
   - 产业政策调整
   - 税收政策变化
   - 出口政策变化

4. **资金风险**
   - 应收账款回收风险
   - 存货积压风险
   - 设备投资回笼慢

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 生产经营 | 生产规模、订单情况、产能利用率、开工率 |
| 原材料采购 | 供应商稳定性、采购价格变化、库存情况 |
| 销售情况 | 主要客户、销售渠道、回款周期、合同签订 |
| 设备状况 | 设备成色、使用年限、维护情况、产能 |
| 环保合规 | 环保手续是否齐全、环保设备是否达标 |
| 员工管理 | 员工稳定性、工资发放、社保缴纳 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 应收账款 | 余额合理性、账龄分析、主要客户占比 |
| 存货 | 原材料、在产品、产成品结构，库存周转 |
| 固定资产 | 原值、折旧、净值、抵押情况 |
| 成本核算 | 主营业务成本结构、毛利率合理性 |
| 经营现金流 | 销售商品收到的现金、购买商品支付的现金 |

#### 交叉验证

- 核对生产记录与销售发票
- 核对原材料采购与库存变动
- 核对员工人数与工资总额
- 核对电费、水费与生产规模匹配度

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 15%-30% |
| 应收账款周转天数 | 60-120天 |
| 存货周转天数 | 60-180天 |
| 资产周转率 | 0.8-2次/年 |
| 应收账款占比 | 20%-40% |
| 存货占比 | 20%-35% |

## 批发零售业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 批发+零售，进货-库存-销售 |
| 资金周转 | 周转速度快，"钱货两清"为主 |
| 季节性 | 受节日、消费季节影响明显 |
| 竞争激烈 | 市场进入门槛低，竞争者众多 |
| 库存管理 | 库存占用资金多，滞销风险大 |
| 利润率低 | 走量为主，毛利率较低 |

### 主要风险点

1. **市场风险**
   - 消费需求变化
   - 竞争加剧，价格战
   - 电商冲击
   - 消费降级

2. **库存风险**
   - 库存积压
   - 过期商品损失
   - 滞销品处理

3. **渠道风险**
   - 供应商关系不稳定
   - 下游客户回款慢
   - 假货风险

4. **经营风险**
   - 店铺租金上涨
   - 人力成本上升
   - 经营合规性

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 销售情况 | 日均销售额、客流量、复购率、客单价 |
| 商品结构 | 主营品类、库存结构、滞销品情况 |
| 供应商 | 供应商数量、合作稳定性、账期 |
| 下游客户 | 主要客户类型、回款情况、账期 |
| 经营场所 | 租赁合同、租金水平、地段人流 |
| 竞争对手 | 周边竞争情况、自身优势 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 库存 | 库存结构、周转率、账龄、减值情况 |
| 应收账款 | 余额合理性、账龄、主要客户 |
| 应付账款 | 供应商账期、拖欠情况 |
| 销售收入 | 进货量、销量、价格匹配度 |
| 现金流 | 销售现金流入、采购现金流出 |

#### 交叉验证

- 核对库存盘点记录与账面存货
- 核对进货发票与库存变动
- 核对销售记录与银行流水
- 核对租金发票与租赁合同

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 15%-25% |
| 存货周转天数 | 30-90天 |
| 应收账款周转天数 | 30-60天 |
| 资产周转率 | 2-5次/年 |
| 存货占比 | 30%-50% |

## 餐饮服务业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 现金收付为主，日结日清 |
| 资金周转 | 周转快，现金流充沛 |
| 季节性 | 受节假日、旅游季节影响 |
| 经营成本 | 食材、人工、租金占比较高 |
| 风险易发 | 食品安全、消防安全、卫生监管 |
| 易模仿 | 口味和服务难以形成壁垒 |

### 主要风险点

1. **经营风险**
   - 食品安全问题
   - 消防安全隐患
   - 卫生监管风险
   - 投诉纠纷风险

2. **市场风险**
   - 客源不稳定
   - 竞争激烈
   - 口味过时
   - 装修老化

3. **成本风险**
   - 食材价格上涨
   - 人工成本上升
   - 租金上涨
   - 水电费用

4. **特殊风险**
   - 股东纠纷
   - 多家连锁管理
   - 垫资经营
   - 民间借贷

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 经营情况 | 上座率、客单价、翻台率、日均营收 |
| 原材料采购 | 供应商、采购频次、价格、新鲜度 |
| 员工管理 | 员工数量、稳定性、工资发放 |
| 卫生安全 | 卫生许可证、消防验收、食品安全培训 |
| 股东情况 | 股东人数、股权比例、是否存在纠纷 |
| 客户评价 | 网络评价、顾客反馈、复购率 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 销售收入 | 客流记录、收银记录、现金流水匹配 |
| 原材料成本 | 采购量、单价、成本占比合理性 |
| 人工成本 | 员工人数、工资水平与营收匹配 |
| 固定成本 | 租金、水电、装修摊销 |
| 现金流 | 日营业额、现金存款记录 |

#### 交叉验证

- 核对收银系统记录与银行流水
- 核对采购发票与食材消耗
- 核对员工工资表与员工人数
- 核对电费、水费与营业规模

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 50%-65% |
| 净利率 | 10%-20% |
| 人工成本占比 | 25%-35% |
| 租金占比 | 10%-20% |
| 食材成本占比 | 30%-40% |

## 建筑装修业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 工程承包、垫资经营为主 |
| 资金周转 | 项目周期长，资金占用大 |
| 季节性 | 受房地产周期、政策影响 |
| 垫资普遍 | 普遍存在垫资现象 |
| 回款风险 | 应收账款回收风险大 |
| 合作复杂 | 往往涉及多方合作 |

### 主要风险点

1. **资金风险**
   - 垫资规模大
   - 回款周期长
   - 资金链断裂风险
   - 民间借贷依赖

2. **市场风险**
   - 房地产周期影响
   - 政策调控风险
   - 行业竞争激烈
   - 项目获取难度

3. **经营风险**
   - 工程质量风险
   - 安全事故风险
   - 分包商管理
   - 材料价格波动

4. **回款风险**
   - 甲方拖欠工程款
   - 质保金回收
   - 工程款结算争议

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 工程项目 | 在建项目、已完成项目、项目合同金额 |
| 资金状况 | 垫资规模、资金来源、回款情况 |
| 合作方 | 主要甲方、分包商、材料供应商 |
| 回款情况 | 工程款结算、质保金、账期 |
| 资质情况 | 资质等级、有效期、安全生产许可证 |
| 施工现场 | 项目进度、质量、安全状况 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 应收账款 | 余额合理性、账龄、主要甲方 |
| 在建工程 | 已完工未结算、已结算未收款 |
| 现金流 | 工程款回收、垫资支出、借款流入 |
| 负债结构 | 银行贷款、民间借贷、应付账款 |
| 收入确认 | 按项目、按进度确认收入 |

#### 交叉验证

- 核对工程合同与应收账款
- 核对项目进度与收入确认
- 核对回款流水与银行记录
- 核对材料采购与项目消耗

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 10%-20% |
| 应收账款周转天数 | 90-180天 |
| 垫资占比 | 30%-60% |
| 应收账款占比 | 30%-50% |
| 资产周转率 | 1-2次/年 |

## 物流运输业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 运输服务、仓储服务 |
| 资金周转 | 周转快，但垫油费普遍 |
| 成本结构 | 油耗、人工、车辆折旧占主要 |
| 风险集中 | 安全事故、车辆事故风险高 |
| 监管严格 | 交通、消防、安全监管 |
| 竞争激烈 | 价格竞争激烈 |

### 主要风险点

1. **安全风险**
   - 交通事故
   - 货物损毁
   - 人员伤亡
   - 安全责任赔偿

2. **成本风险**
   - 油价波动
   - 车辆维修保养
   - 人工成本上升
   - 路桥费上涨

3. **经营风险**
   - 车辆超载被罚
   - 驾驶证扣分
   - 车辆年检
   - 执法查处

4. **市场风险**
   - 运价波动
   - 订单不稳定
   - 竞争加剧

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 车辆状况 | 车辆数量、成色、年检情况 |
| 运输业务 | 运输线路、单次运输量、运价 |
| 客户情况 | 主要客户、合作稳定性、回款情况 |
| 司机管理 | 司机数量、驾驶证、工资发放 |
| 安全记录 | 事故记录、违章记录、保险情况 |
| 加油管理 | 油卡、加油量、油耗 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 应收账款 | 余额合理性、账龄、主要客户 |
| 油耗成本 | 油耗量、单价、占总成本比例 |
| 车辆折旧 | 车辆原值、折旧年限、净值 |
| 收入确认 | 运输收入与业务量匹配 |
| 现金流 | 运费收入、油费支出、其他支出 |

#### 交叉验证

- 核对运输单据与收入确认
- 核对加油记录与油耗成本
- 核对车辆年检与车辆状况
- 核对违章记录与安全状况

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 15%-25% |
| 应收账款周转天数 | 30-60天 |
| 油耗成本占比 | 30%-40% |
| 应收账款占比 | 15%-30% |

## 教育培训业

### 行业特征

| 维度 | 特征描述 |
|------|----------|
| 经营模式 | 预收费、服务型 |
| 资金周转 | 预收学费，现金流充沛 |
| 政策敏感 | 教育政策、培训政策影响大 |
| 监管严格 | 教育局、市场监管多部门监管 |
| 家长关注度高 | 口碑传播快 |
| 师资依赖 | 优质师资是核心竞争力 |

### 主要风险点

1. **政策风险**
   - 培训资质政策变化
   - 收费监管政策
   - 校外培训整顿
   - 减负政策影响

2. **经营风险**
   - 教师流失
   - 课程质量
   - 家长投诉
   - 竞争加剧

3. **合规风险**
   - 资质不齐全
   - 超范围经营
   - 收费不规范
   - 广告违规

4. **资金风险**
   - 预收费规模
   - 资金挪用风险
   - 跑路风险

### 调查要点

#### 现场调查

| 调查项 | 重点内容 |
|--------|----------|
| 办学资质 | 办学许可证、营业执照、消防验收 |
| 课程设置 | 课程种类、收费标准、课时安排 |
| 师资队伍 | 教师数量、资质、稳定性 |
| 学生情况 | 在读学生数、新生招生、续费率 |
| 教学场地 | 场地面积、装修、教学设备 |
| 家长评价 | 网络评价、投诉情况、口碑 |

#### 财务核查

| 核查项 | 关键点 |
|--------|--------|
| 预收账款 | 余额合理性、账龄、退费情况 |
| 收入确认 | 按课时确认收入与预收匹配 |
| 成本结构 | 师资成本、场地租金、其他成本 |
| 现金流 | 学费收入、退费支出、其他支出 |
| 资金使用 | 预收费资金使用情况 |

#### 交叉验证

- 核对学生人数与预收学费
- 核对课时记录与收入确认
- 核对教师工资表与师资人数
- 核对场地租赁与租金支出

### 关键财务指标参考

| 指标 | 参考区间 |
|------|----------|
| 毛利率 | 40%-60% |
| 净利率 | 15%-30% |
| 师资成本占比 | 30%-50% |
| 场地租金占比 | 15%-25% |
| 预收账款占比 | 20%-40% |

## 使用说明

1. **行业选择** - 根据客户主营业务选择对应的行业模板
2. **重点关注** - 结合客户特点，重点关注相关风险点
3. **灵活调整** - 根据实际情况调整调查要点和指标参考
4. **持续更新** - 建议定期更新行业数据和风险特征
5. **交叉验证** - 充分利用交叉验证方法，提高调查质量

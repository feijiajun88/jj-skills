# 行业调查指引技能

## 技能概述

本技能为普道小微贷款信贷员提供**贷前行业调查**的标准化指引和报告生成工具。

通过整合内置行业知识库与实时联网搜索，快速输出结构化的行业调查报告，帮助信贷员：
- 了解目标行业的市场行情与发展趋势
- 识别行业风险特征与关键风险点
- 明确现场调查重点与注意事项
- 参考行业关键指标与财务基准

---

## 核心功能

### 1. 行业知识库（内置）

涵盖常见小微行业的基本特征、风险点、调查要点：

- **制造业**：加工制造、轻工制造、纺织服装
- **批发零售**：日用百货、建材、农资、电子产品
- **餐饮服务**：餐厅、酒店、快餐连锁
- **建筑装修**：建筑施工、装饰装修、工程承包
- **物流运输**：货运、快递、仓储服务
- **教育服务**：培训机构、幼儿园
- **其他服务**：美容美发、家政服务、娱乐场所

### 2. 实时联网搜索

利用联网搜索获取最新行业信息：
- 行业政策与监管动态
- 市场规模与发展趋势
- 竞争格局与头部企业
- 近期行业新闻与风险事件

### 3. 调查清单生成

根据行业特征自动生成：
- 现场调查要点清单
- 财务核查重点
- 风险预警指标
- 交叉验证方法

### 4. 结构化报告输出

生成标准Markdown格式报告，包含：
- 行业概况与市场分析
- 行业风险特征评估
- 调查要点与操作指引
- 财务指标参考值
- 风险提示与建议

## 使用方法

### 基本用法

```
输入：请帮我调查[行业名称]的行业情况

示例：
- 请帮我调查制造业的行业情况
- 我想了解餐饮行业的风险特征
- 批发零售业需要关注哪些调查要点
```

### 高级用法

```
输入：请帮我调查[行业名称]，重点关注[特定方面]

示例：
- 请帮我调查餐饮业，重点关注疫情后恢复情况
- 调查制造业，重点关注环保政策和用工成本
- 调查装修行业，重点关注垫资情况和回款风险
```

### 查询流程

1. **识别行业** - 根据用户输入确定目标行业
2. **知识库检索** - 从内置模板库中提取行业调查要点
3. **联网搜索** - 搜索该行业的最新信息（政策、数据、新闻）
4. **综合分析** - 结合知识库与实时信息，生成行业分析
5. **输出报告** - 按照标准模板生成结构化Markdown报告

```
用户输入
    ↓

┌──────────────┬──────────────┐
│ 内置知识库    │   联网搜索    │

└──────────────┴──────────────┘
    ↓
综合分析与整合
    ↓
生成结构化报告
    ↓
输出Markdown文档
```

### Token管理（重要）

为保持主上下文整洁，联网搜索应通过Task子代理执行：

- Task代理执行搜索并处理结果
- 返回精简的JSON或简短Markdown摘要
- 主上下文综合知识库与搜索结果生成报告

## 报告结构

生成的报告包含以下模块：

### 一、行业概况
- 行业定义与分类
- 市场规模与发展趋势
- 竞争格局与主要参与者
- 行业政策与监管环境

### 二、风险特征分析
- 行业周期性分析
- 主要经营风险
- 行业政策风险
- 其他特殊风险

### 三、信贷调查要点
- 现场调查重点
- 财务核查关键点
- 交叉验证方法
- 客户配合度评估

### 四、关键财务指标参考
- 行业毛利率区间
- 资产周转率参考
- 应收账款周转天数
- 行业盈利能力指标

### 五、风险提示与建议
- 重点风险提示
- 贷前调查建议
- 贷后管理要点

## 使用场景

本技能适用于以下场景：

1. **贷前准备** - 在实地调查前快速了解目标行业情况
2. **风险识别** - 识别特定行业的潜在风险点
3. **新员工培训** - 帮助新信贷员掌握行业调查方法
4. **标准化作业** - 推动信贷调查流程的标准化和规范化
5. **跨区域展业** - 了解不同地区的行业特征和差异

---

## 注意事项

1. **报告仅供参考** - 本技能生成的报告仅供参考，实际调查需结合客户具体情况
2. **持续更新** - 行业情况变化较快，建议定期更新内置知识库
3. **数据时效性** - 联网搜索的信息可能存在滞后，需注意数据发布时间
4. **地区差异** - 同一行业在不同地区可能存在显著差异，需考虑地域因素
5. **保密要求** - 使用联网搜索时注意保护客户隐私和商业机密
